# Useful Tools Mod — 1.21.1 Fabric

A Minecraft 1.21.1 Fabric mod that adds a wide selection of tools, weapons, and armor sets built from reinforced versions of common materials, plus a handful of unique items, blocks, and a friendly Ghost mob.

## Features

- **Reinforced & polished tool/armor sets** for emerald, redstone, glowstone, gold, lapis, obsidian, ectoplasm, coal, and more — each with its own tier, durability, and effects.
- **Edible food-themed gear** (cake, golden apple, bread, melon, sweet berry, pumpkin pie, mushroom, pufferfish, honey, chorus fruit, dried kelp, rotten flesh) — right-click to eat from your hotbar.
- **Vanilla-material tool & armor effects** — paper, feather, glass, sponge, nether wart, dripstone, cactus, bone, nether brick, magma cream, slime, blaze, purpur, ghast tear, eye of ender, shulker, echo shard, dragon breath, phantom, turtle scute, nautilus, and more.
- **Spectral Infuser** block — combine Spectral armor with ectoplasm to craft Spectral Infused armor that turns Ghosts neutral.
- **Ghosts** — a friendly flying mob that can be tamed with ectoplasm, follows the player, and only takes damage from ecto weapons.
- **Custom ores & worldgen** — Reinforced Gold ore (overworld, deepslate, nether, end variants), plus storage blocks for every new material.

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.1.
2. Install [Fabric API](https://modrinth.com/mod/fabric-api) (`>= 0.116.9+1.21.1`).
3. Drop `usefultoolsmod-2.2.1-1.21.1-fabric.jar` into your `mods/` folder.

### Optional integrations

- [Just Enough Items (JEI)](https://modrinth.com/mod/jei) — Spectral Infuser recipes appear in JEI.
- [WTHIT](https://modrinth.com/mod/wthit) — Spectral Infuser shows progress in the WTHIT tooltip.

## Requirements

- Minecraft 1.21.1
- Fabric Loader >= 0.15.0
- Fabric API
- Java 21
